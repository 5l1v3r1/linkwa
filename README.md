# Create Link Whatsapp (python)
Simple create link whatsapp otomatis, dapat dijalankan di windows, linux, termux, dan macos. 

# Install
- apt update && apt upgrade 
- apt install python git
- git clone https://github.com/kyo1337/linkwa
- cd linkwa
- python linkwa.py
